ItemBuilder

class ItemBuilder(itemStack: ItemStack)

Represents a builder for creating ItemStack objects.

The ItemBuilder class provides methods to conveniently set various properties of an ItemStack object, such as display name, lore, amount, custom model data, and localized name.
