UndefinedCommand

abstract class UndefinedCommand(commandName: String, commandType: CommandType = CommandType.ALL, description: String = "Command created with UndefinedAPI", aliases: List<String> = emptyList(), permission: String = "") : BukkitCommand

Abstract class representing an undefined command in Bukkit.

Parameters

commandName

The name of the command.

commandType

The type of the command. Default is CommandType.ALL.

description

The description of the command. Default is "Command created with UndefinedAPI".

aliases

The aliases for the command. Default is an empty list.

permission

The permission required to execute the command. Default is an empty string.
