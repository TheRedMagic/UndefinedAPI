event

inline fun <T : Event> event(priority: EventPriority = EventPriority.NORMAL, ignoreCancelled: Boolean = false, crossinline callback: T.() -> Unit): UndefinedListener

Registers an event listener for a specific type of event with the given priority and cancellation options.

Return

An instance of UndefinedListener that can be used to unregister the event listener.

Parameters

T

The type of event to listen for.

priority

The priority of the event listener. Default value is EventPriority.NORMAL.

ignoreCancelled

Determines whether cancelled events should be ignored. Default value is false.

callback

The lambda function that will be executed when the event occurs. It takes an instance of the event as a receiver and does not return any value.
